# М-Тех Гео — схемы

Схемы дашборда и лендинга: продукт удержания клиентской базы для салонов на YClients.
Гео — вход и объяснение, YClients — деньги.

Открыть: `index.html` в браузере.

Цифры в макетах — из живого кабинета работающего салона. Название, имена клиентов
и мастеров заменены. Что настоящее, а что заглушка — в первом блоке страницы.
