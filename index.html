Правила, которые нельзя нарушать при вёрстке

  1. Никаких прогнозов в интерфейсе. Только «вернулось 7 из 31» — факт, который нельзя оспорить. Обещание «3–4 записи из 8» в текущем продукте меняется на «5 из 8» от смены фильтра периода. Это генерация, а не расчёт.
  2. Одна серия, на деньгах. «4 недели роста» висит только на перезаписи, потому что её нельзя накрутить. Серия на заполненности карточки или на выполненных задачах превратит продукт в тренажёр по нажиманию кнопок.
  3. Задача — то, что делается из кресла за 30 секунд. «Напишите Марии, текст готов» — задача. «Установите звукопоглощающие панели» — вывод для владельца, ему место в телеграм-сводке. Смешение двух классов даёт 0 из 19.
  4. Каждая цифра кликается в живой список. Составных индексов нет: цифра, которую нельзя перепроверить руками, становится нашей выдумкой и рушит доверие к остальным.
  5. Ролей нет. Один экран для всех, владельцу — сводка в телеграм. Если в сводке нет цифры денег, она не отправляется.
  6. Соседи — только в платном. Бесплатно отдаём то, что про него самого: карточка, заполненность, хвалят и жалуются. Ранг из API Яндекса хранить запрещено лицензией, конкуренты берутся из 2ГИС в платном контуре.
