Разрывы отвечают «почему», задачи — «что делать». Кнопки действия живут в блоке 04, здесь только провал в данные.

Правила, которые нельзя нарушать при вёрстке

  1. Никаких прогнозов в интерфейсе. Только «вернулось 7 из 31». Обещание «3–4 записи из 8» в текущем продукте меняется на «5 из 8» от смены фильтра периода — это генерация, а не расчёт.
  2. Одна серия, на деньгах. «4 недели роста» висит только на перезаписи, потому что её нельзя накрутить. Серия на заполненности карточки или на выполненных задачах превратит продукт в тренажёр.
  3. Задача — то, что делается из кресла за 30 секунд. Каждая несёт источник, деньги и результат. Четыре в день, а не девятнадцать.
  4. Если задачу можно сгенерировать без CRM — она не наша. «Ответьте на отзыв» умеет кто угодно за 990 ₽.
  5. Каждая цифра кликается в живой список. Составных индексов нет.
  6. Ролей нет. Один экран для всех, владельцу — сводка в телеграм. Если в сводке нет цифры денег, она не отправляется.
  7. Соседи — только в платном. Ранг из API Яндекса хранить запрещено лицензией; конкуренты берутся из 2ГИС в платном контуре.
